Skywork Notes is a lightweight, flexible knowledge and context workspace designed to capture ideas, references, and task-related information as you work. Whether you’re browsing, reading, or organizing materials, notes help consolidate insights into a reusable context layer that can be applied across projects.

![note](https://static-us-img.skywork.ai/test/note/2025-12-12/6241771231207272527/1999339778030030848_note.image/png "note")



### **Using Notes as Dynamic Context for Agent Generation**

Notes act as a live context source for Agent-assisted tasks. As you update research materials, add metrics, or log project progress, the Agent automatically incorporates the latest note content into its reasoning and output resulting in responses that are more current, more aligned with your intent, and better tailored to the task at hand.

- **Providing background**  
Notes allow you to record requirements, constraints, references, and other foundational information before starting a project, ensuring the Agent operates from a shared and consistent understanding.


- **Adding evolving information**  
New insights from reading, meetings, or ongoing analysis can be added at any time. Notes maintain continuity, enabling outputs that stay synchronized with the most recent information.


- **Storing reusable knowledge**  
Definitions, data points, recurring explanations, and analytical frameworks can be preserved in notes for repeated use across multiple projects.

## Connecting Notes to Projects

### @-mentioning notes from the Home input box

From the Home page, you can reference a note directly using **@note name**.The note is immediately added as contextual input for the task, enabling quick access to existing materials without needing to navigate to a project or note page.

![image2](https://static-us-img.skywork.ai/test/note/2025-12-12/6241771231207272527/1999340166418386944_image2.image/png "image2")



### Starting or connecting a project from within a note

A project can be created or associated directly from a note. From the upper-right corner of the note page, select **“Project” button**. The note then becomes part of the project’s contextual foundation.Subsequent projects initiated from the note automatically use it as the initial context.

![Area1](https://static-us-img.skywork.ai/prod/note/2025-12-15/4818646732570429176/2000504644260409344_Area1.gif.image/gif "Area1")



### Adding notes from within a project

Inside a project, notes can be added through the left resource panel.Associated notes appear in the project’s resource list and are included in all future generations. Updates to a note are immediately reflected in how the Agent interprets and executes tasks.

# **Editing with Markdown & Shortcuts**

Use simple Markdown syntax to create headings, bold text, lists, and more. Shortcuts help you write and format your content efficiently.


| **Style**          | **Markdown Syntax**                                       | **Shortcut**                  |
| ------------------ | --------------------------------------------------------- | ----------------------------- |
| # **Headings**     | Use `# + space`; the number of `#` sets the heading level | cmd/ctrl + option/shift + 1–4 |
| **Bold**           | Wrap text with `**` or `__` + space                       | cmd/ctrl + B                  |
| *Italic*           | Wrap text with `*` or `_` + space                         | cmd/ctrl + I                  |
| ==Highlight==      | Wrap text with `==` + space                               | —                             |
| ++Underline++      | Wrap text with `~` + space                                | cmd/ctrl + U                  |
| ~~Strikethrough~~  | Wrap text with `~~` + space                               | —                             |
| 1. Ordered List    | Number + `.` + space                                      | cmd/ctrl + option/shift + 7   |
| - Bullet List      | `-`, `*`, or `+` + space                                  | cmd/ctrl + option/shift + 8   |
| - [ ] Task List    | `[]` + space                                              | cmd/ctrl + option/shift + 9   |
| ``` Code Block ``` | Three backticks “```” + space                             | —                             |
| > Quote            | `>` + space                                               | —                             |


## **Using the “/” Command Menu**

Pressing `/`  brings up a full menu of content blocks you can choose to insert. You can shortcut this further by pressing `/` and then typing what you want. Here are your choices:

- `/text` creates a new text block.


- `/h1`  creates a large heading.


- `/h2` creates a medium-sized heading.


- `/h3`  creates a small heading.


- `/bullet` creates a bulleted list.


- `/num` creates a numbered list.


- `/todo` creates a to-do list with checkboxes.


- `/quote` creates a quote block of larger text.


- `/image`  inserts an image block.


- `/sep` inserts a horizontal separator.  


- `/table`  inserts a table block.
