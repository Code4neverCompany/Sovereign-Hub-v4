# 🛠 4NeverCompany - Tech Stack Definition

Dieser Tech-Stack ist der verbindliche Standard für alle Software-Artefakte der **4NeverCompany**. Er wurde gewählt, um technische Exzellenz, Sicherheit und plattformübergreifende Konsistenz zu gewährleisten.

## 🌐 Web & Core
*   **Framework:** `Next.js` (App Router) – Für maximale Performance und SEO.
*   **Language:** `TypeScript` – Strikte Typisierung für robuste Codebasen.
*   **Runtime:** `Node.js` (LTS) / `Bun` (für High-Performance Tasks).

## 🎨 UI & Design (The Code4Never Aesthetic)
*   **Styling:** `Tailwind CSS` – Zur präzisen Umsetzung der #000000 / #C0A060 Farbpalette.
*   **Components:** `Shadcn/UI` & `Magic UI` – Moderne, anpassbare Komponenten.
*   **Animations:** `Framer Motion` – Für flüssige, kybernetische Interaktionen.
*   **Icons:** Lucide React (angepasst an die ∞4-Doktrin).

## 🔐 Identity & Security
*   **Authentication:** `Clerk` – Der Standard für sicheres User-Management und Auth-Flows (Customized Dark Theme).

## 🖥 Desktop & Mobile
*   **Desktop:** `Tauri` – Zur Konvertierung von Web-Artefakten in native Apps für Windows, macOS und Linux.
*   **Mobile:** PWA (Progressive Web App) Support standardmäßig.

## 🗄 Data & Backend
*   **Database:** `Supabase` / `PostgreSQL` – Skalierbar und zuverlässig.
*   **ORM:** `Prisma` oder `Drizzle` – Für typsicheren Datenbankzugriff.
